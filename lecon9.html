IX. Indexation négative

Explication :

Python permet aussi d'utiliser des indices négatifs pour accéder depuis la fin chaîne :
- -1 fait référence au dernier caractère
- -2 fait référence à l'avant dernier caractère
- Et ainsi de suite...

Exemple :

nom = N O C T I S
index = -6 -5 -4 -3 -2 -1

Pour accéder à un caractère spécifique, on utilise [ ]
avec le numéro d'index.
