II. Variables

Explication :

Quand tu as besoin de stocker des informations pour les utiliser plus tard.
Par exemple, de l'argent ou de la nourriture.
Pour faire ça tu vas utiliser des variables.

Une variable est un espace dans la mémoire de ton appareil qui stocke les informations.
Plus précisément , 3 informations :

- Un Nom → comment tu l'appelle
- Un Type → quel type de données elle contient
- Une Valeur → la donnée qu'elle contient
