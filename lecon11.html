XI. Le découpage de chaîne plus intélligemment

Explication :

Le langage python fournit des "raccourcis" pour le découpage de chaînes :
- Si tu oublie de mettre l'indice du début, python utilise 0 comme début par défaut.
- Si tu oublie de mettre l'indice de fin, python utilise la longueur de la chaîne par défaut.

Exemple :

nom = "informatique"
premiere_partie = nom[:5]

nom = "informatique"
deuxieme_partie = nom[5:]
