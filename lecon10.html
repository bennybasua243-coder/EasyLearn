X. Introduction au découpage de chaîne (slicing)

Explication :

Le découpage de chaînes (slicing) est une façon d'extraire une partie d'une chaîne (une sous-chaîne). C'est comme couper un morceau.
La syntaxe de base pour le découpage est :
string[début:fin] Dans le quel :
- début estv l'indice où commence le découpage(inclus)
- fin est l'indice où se termine le découpage (exclus)

Exemple :

nom = "informatique"
premiere_partie = nom[0:5]
